Webviz generated schemas

If you don't want to regularly run "webviz schema" in order to update your YAML validation schema for use in e.g. Visual Studio Code, this site hosts auto-generated schemas that are updated nightly.

Simply copy the URL below which includes the plugin projects you are using into e.g. Visual Studio Code settings, and you will automatically have a validation schema always using the latest releases of the included plugin projects.

See Webviz documentation for more details on how to configure Visual Studio Code.
