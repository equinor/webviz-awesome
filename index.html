Webviz generated schemas

If you don't want to regularly run "webviz schema" in order to update your JSON/YAML schema for use in e.g. Visual Studio Code, this site hosts auto-generated schemas that are updated nightly.

Simply copy the URL below which equals the plugin projects/environment you are using into e.g. Visual Studio Code settings, and we will update it for you.

  • Latest PyPI releases of plugin projects used in Equinor Komodo: https://equinor.github.io/webviz-awesome/schemas/webviz-schema-komodo.json
